This page appears at the top of the left menu in French language, only for the Admin group. 
It summarizes the settings made on wiki.js.
 

Groups

  • Guests  : System-locked group, only allows reading and is the default when creating an account on the wiki,
  • Admin : System-locked group giving full access,
  • Contributor : Group created for contributors. Gives write access to all pages (page rules) as well as to the navigation part.

Group changes can only be made by members of the Admin group.
