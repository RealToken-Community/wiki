Wiki's GitHub

The Wiki's database (PostgreSQL) is replicated, bidirectionally, on a Community GitHub Repository.

It is therefore possible to update the Wiki by submitting a Pull Request on this Repo (preprod branch).
This approach requires knowledge of how GitHub works, which is often the case for developers.

The replication runs every hour.

 

File Formats

Files edited from the Wiki:

  • with Visual Editor: are HTML files,
  • with Markdown: are MD files.

these formats must be maintained, otherwise certain page presentation modes may be lost.

Folder Organization

The wiki is available in three languages (en, fr, es, pt).
The default language is English, so the corresponding files are found at the root.
For other languages (French, Spanish and Portuguese), a specific folder contains the corresponding pages.

Images and documents are stored in the corresponding language folder.
