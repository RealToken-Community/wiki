Grupos

  • Convidados: Grupo bloqueado pelo sistema, apenas permite a leitura e é o padrão quando se cria uma conta wiki,
  • Admin: Grupo bloqueado pelo sistema dando acesso total,
  • Contribuidor: Grupo criado para os contribuidores. Dá acesso de escrita a todas as páginas (regras de página), bem como à parte de navegação.

As alterações no grupo só podem ser feitas por membros do grupo Admin.
