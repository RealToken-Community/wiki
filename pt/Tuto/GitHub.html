Wiki GitHub

 

A base de dados Wiki (PostgreSQL) é replicada, bidirecionalmente, num Repositório GitHub da Comunidade.

Portanto, é possível atualizar o Wiki enviando um pull request para este repositório (ramo de pré-produção). Esta abordagem requer saber como funciona o GitHub, o que é geralmente o caso dos programadores.

A replicação é executada de hora em hora.

 

Formato de ficheiro

Ficheiros editados da Wiki:

  • com Editor Visual: são ficheiros HTML,
  • com Markdown: são ficheiros MD.

Estes formatos devem ser mantidos, caso contrário perderá determinados modos de apresentação da página.

Organização de arquivos

O wiki está disponível em três línguas (en, fr, es, pt).
O idioma predefinido é o inglês, pelo que os ficheiros correspondentes estão na raiz. 
Para os restantes idiomas (francês, espanhol e português), uma pasta específica contém as páginas correspondentes.

As imagens e os documentos são armazenados na pasta do idioma correspondente.
