GitHub del Wiki

La base de datos (PostgreSQL) del Wiki se replica, de forma bidireccional, en un Repositorio GitHub de la Comunidad.

Por lo tanto, es posible actualizar el Wiki enviando una Pull Request a este Repositorio (rama preprod).
Este enfoque requiere conocer el modo de funcionamiento de GitHub, lo cual es a menudo el caso para los desarrolladores.

La replicación se ejecuta cada hora.

 

Formato de los archivos 

Los archivos editados desde el Wiki:

  • con Visual Editor: son archivos HTML,
  • con Markdown: son archivos MD.

estos formatos deben conservarse, so pena de perder ciertos modos de presentación de la página.

Organización de las carpetas

El wiki está disponible en cuatro idiomas (en, fr, es, pt).
El idioma predeterminado es el inglés, por lo que los archivos correspondientes se encuentran en la raíz.
Para los otros idiomas (Francés, Español y Portugués), una carpeta específica contiene las páginas correspondientes.

Las imágenes y documentos se almacenan en la carpeta correspondiente del idioma.
