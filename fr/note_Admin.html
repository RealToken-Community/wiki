Cette page n'apparait en tête du menu de gauche en langue Fr, que pour le groupe Admin. 
Elle synthétise les paramétrages faits sur wiki.js.
 

Groupes

  • Guests  : Groupe bloqué par le système, ne permet que la lecture et est celui par défaut lors de la création d'un compte sur le wiki,
  • Admin : Groupe bloqué par le système donnant tous les accès,
  • Contributor : Groupe créé pour les contributeurs. Donne accès en écriture à toutes les page (page rules) ainsi qu'à la partie navigation.

Les changements de groupe ne peuvent être effectués que par les membre du groupe Admin.

 

Autres paramétrages

Par page du menu Admin : 

  • Général :
    • Désactivés : Comment et Edit shortcuts
  • Locale
    • Langue par défaut : Anglais (Ne pas modifier, car les sauvegardes de la langue correspondantes vont à la racine, ce qui désorganise les dossier et réplications Git).
  • Theme :
    • Dark mode : désactivé,
    • Table of content : mise à droite,
    • code injection : ajouté par Mickael
  • Analytics : Matomo, Site ID : 2, Server Host : https://data.realt.community
  • Authentification : Local avec Self-registration, assignée au groupe Guests
  • Rendering : Markdown> Html : tout activé
  • Search Engine : DB PostgreSQL, Dictionary : English
  • Storage : Git , Url : git@github.com:RealToken-Community/wiki.git, Branch : preprod, Auth par SSH, Bidirectional, Délais : 1h
    Fonctions pour : forcer la synchro, purger le repo local, …
  • Mail : Sender Name : Devs RealToken email : dev@realt.community, Host : mail.infomaniak.com
  • Utilities : nombreuses fonction d'admin technique..
