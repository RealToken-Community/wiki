La base de données (PostgreSQL) du Wiki est répliquée, de façon bidirectionnelle, sur un Repository le GitHub de la Communauté.

Il est donc possible de faire des mise à jour du Wiki, en soumettant une pull Request sur ce Repo (branche preprod).
Cette approche nécessite de connaitre le mode de fonctionnement de GitHub, ce qui est souvent le cas pour les développeurs.

 

 

 

 

 

 

 

 

 

 

Les fichiers édités à partir du Wiki :

  • avec Visual Editor : sont des fichiers HTML,
  • avec Markdown : sont des fichiers MD.

ce format est à conserver, sous peine de perdre certains modes de présentation de la page.
